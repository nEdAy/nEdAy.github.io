Hexo系列教程：基础篇（3）Hexo域名绑定和域名解析

发表于 2018-03-02
字数统计: 676 | 阅读时长 ≈ 2

前言

简介

本篇教程讲解如何注册购买域名（高校学生获得免费域名）并将该域名绑定解析到自己的Hexo托管部署平台。

正文

域名注册

免费cn域名

所以有全日制统招在读高校学生，在完成腾讯云平台的学生认证后，在大学毕业时间前可享受云+校园计划的优惠政策。

优惠内容：1元=服务器+免费域名

服务器：

1核1G 1M带宽／免费赠送系统盘

Linux 20G ／ Windows 50G 按月扶持

域名：

在校期间可获得.cn域名一个，按年扶持

申请流程：

1注册腾讯云帐号

2实名认证（财付通认证）

3学生认证

4获取福利

活动地址、详细指引

付费域名

推荐购买注册.com或.cn等顶级域名；

推荐选择国内万网或者国外的Goddady购买注册，安全，而且可以使用支付宝。

域名绑定

先将自己域名绑定第三方托管平台的Page服务，为下一步域名解析做准备

GitHub需要在Hexo本地根目录中的source文件夹里创建名称为“CNAME”的文件，里面添加你的域名信息，如：neday.cn

注意：

1. “CNAME”的文件不带任何后缀；
2. 注意是“CNAME”不是“CHAME”；
3. 域名前面不添加http://，如果二级域名是www则省略，是blog则为blog.neday.cn。

Coding则需要登录系统，并手动在 博客项目——Pages 服务——绑定一个自定义域名 下输入你的域名，输入规则同上。

最后，输入命令进行托管部署的更新上传：

hexo d -g

域名解析

如果你选择的是腾讯云学生免费域名，则可使用腾讯云云解析；

如果你选择的是万网注册的域名，可以使用其自带的域名解析服务；

此外也推荐选择免费的DNSPod国内版、DNSPod国际版

以腾讯云云解析为例，说明需要注意以下几点：

1.分线路，实现国内默认线路访问跳转到Coding，国外线路访问跳转到GitHub；
2.如二级域名为www则主机记录为www，为blog则主机记录为blog；
3.记录类型必须为CNAME，记录值后有.；

主机记录 记录类型 线路 记录值
blog CNAME 默认 neday.coding.me.
blog CNAME 国外 nEdAy.github.io.

至此，就可以使用自己的域名来访问博客了。
